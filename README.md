# JEAF Security API README #

This repository contains the source code of JEAF Security API. JEAF Security provides all functionality that is needed for authentication and authorization in enterprise applications.

## Links ##
For further information please refer to:

* [JEAF Security](https://anaptecs.atlassian.net/l/c/c6DSCPAq)
* [Maven Repository](https://search.maven.org/artifact/com.anaptecs.jeaf.security/jeaf-security-api)
* [Javadoc JEAF Authorization Service Provider API](https://javadoc.io/doc/com.anaptecs.jeaf.security/jeaf-authorization-service-provider-api/latest/index.html)
* [Javadoc JEAF Identity Service Provider API](https://javadoc.io/doc/com.anaptecs.jeaf.security/jeaf-identity-service-provider-api/latest/index.html)

## How do I get set up? ##

* Create a clone of this repository on your local machine.
* Execute Maven on the top level project `maven clean install`
